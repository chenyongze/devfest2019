# DevFest Beijing 2019

时间：2019 年 12 月 8 日全天

地点：北京新世纪日航饭店

概述：全天上午两个分会场(Web、机器学习)、下午两个分会场(移动开发、云计算)，共 18 位讲师进行分享。
